# 🏀 バスケコーチくん

バスケットボールの試合中にiPhoneのカメラで映像を解析し、攻撃・守備の切り替わりを自動検出してBluetoothイヤホンに音声で指示するツールです。

---

## 必要なもの

- **iPhone**（Safariで動作）
- **Bluetoothイヤホン**（ワイヤレスイヤホン）
- **Gemini APIキー**（Google AI Studio で無料取得）

---

## Gemini APIキーの取得方法

1. [Google AI Studio](https://aistudio.google.com/app/apikey) を開く
2. 「Create API key」をクリック
3. 表示された `AIza...` から始まる文字列をコピーしておく
4. 無料枠：Gemini 2.0 Flash は1日1500リクエストまで無料

---

## 使い方

### 1. アプリを開く

iPhoneのSafariで以下のURLを開く：

```
https://hirokunimen0705-glitch.github.io/basketball-coach/
```

### 2. APIキーを入力・保存

- 「Gemini APIキー」欄に取得したAPIキーを貼り付ける
- 「保存」ボタンを押す（次回から自動入力される）

### 3. 設定を調整（任意）

| 設定項目 | 説明 | デフォルト |
|---------|------|-----------|
| 解析間隔 | 何秒ごとにカメラ画像を解析するか | 2秒 |
| 攻撃時の指示 | 攻撃に切り替わったときにイヤホンから流れる言葉 | 「攻撃です。シュートを狙って。」 |
| 守備時の指示 | 守備に切り替わったときにイヤホンから流れる言葉 | 「守備に切り替え。マークにつけ。」 |

### 4. コートサイドで固定・開始

1. iPhoneをコートが見える位置に固定する（三脚など）
2. Bluetoothイヤホンを接続する
3. 「▶ 開始」を押す
4. カメラ使用の許可を求められたら「許可」を選ぶ
5. 自動で解析が始まり、攻守が切り替わるとイヤホンに音声が流れる

### 5. 停止するとき

「■ 停止」ボタンを押す → 設定画面に戻る

---

## 画面の見方

```
[ ⚡ 攻撃 ]  ← 現在の状態（攻撃 or 守備）        [ ■ 停止 ]

[ カメラ映像 ]

[ 解析ログ ]
  12:30:01 ▶ 開始
  12:30:03   攻撃継続
  12:30:05 🔄 → 守備          ← 切り替わり検出（同時に音声も流れる）
```

- **切り替わりのときだけ**音声が流れる（同じ状態が続いても繰り返さない）
- ログは最大50件表示

## ホーム画面に追加する方法（PWA）

Safariの共有ボタン → 「ホーム画面に追加」で、アプリのように使える。

## よくある質問

**Q. 音が出ない**
A. 「▶ 開始」ボタンを押した瞬間に音声がアンロックされる。マナーモードでも鳴る（SpeechSynthesis API はマナーモード非依存）。Bluetoothイヤホンが接続されているか確認。

**Q. カメラが起動しない**
A. HTTPS（GitHub Pages の URL）で開いていることを確認。`http://` では動かない。

**Q. 攻守の判定がズレる**
A. 解析間隔を短く（1秒）するか、カスタム指示を試合に合わせて変える。カメラをコート全体が映る位置に固定すると精度が上がる。

**Q. 「HTTP 429」エラーが出る**
A. Gemini API の無料枠（1日1500リクエスト）を超えた。解析間隔を長くするか、翌日に使う。

**Q. APIキーが毎回消える**
A. 「保存」ボタンを押せば次回から自動入力される（ブラウザのローカルストレージに保存）。Safariの「履歴とWebサイトデータを消去」を実行すると消える。

## 技術仕様（参考）

- **AI**: Gemini 2.0 Flash（無料枠）
- **画像解析**: 2秒ごとにカメラ映像をキャプチャ → 320×180 JPEG に縮小 → Gemini APIに送信
- **音声**: Web Speech API（iOS Safari 対応）
- **動作環境**: iPhone Safari / Android Chrome（HTTPS必須）
